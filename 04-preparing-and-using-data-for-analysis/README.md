# Section 4: Preparing and Using Data for Analysis

**Exam Weight: ~25%**

One of the highest-weighted sections. Covers transforming raw data into analysis-ready
formats for visualization tools, ML/AI use cases, and sharing with consumers.

## Subsections

| Subsection | Topic | File |
|------------|-------|------|
| 4.1 | [Preparing data for visualization](./4.1-preparing-data-for-visualization.md) | Security, masking, IAM, Cloud DLP |
| 4.2 | [Preparing data for AI and ML](./4.2-preparing-data-for-ai-and-ml.md) | Feature engineering, training/serving, RAG, embeddings |
| 4.3 | [Sharing data](./4.3-sharing-data.md) | BigQuery Analytics Hub |

## Key Themes

- **Data security for analysis**: Apply masking, column-level security, and DLP
  before exposing data to visualization tools
- **ML readiness**: Feature engineering, BigQuery ML, Vertex AI feature stores
- **Generative AI patterns**: Embeddings, RAG, and unstructured data preparation
- **Data sharing**: Analytics Hub for secure, governed data product distribution
