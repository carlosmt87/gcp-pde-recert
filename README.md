# GCP Professional Data Engineer Renewal — Study Repository

## Exam Overview

The **Professional Data Engineer Renewal** exam validates that a data engineer can design,
build, operationalize, secure, and monitor data processing systems on Google Cloud. It
emphasizes the latest GCP-managed services, AI/ML data preparation, and governance tooling.

- **Format:** Multiple choice / multiple select, scenario-based questions
- **Duration:** 2 hours
- **Recommended experience:** 3+ years industry experience, 1+ year GCP
