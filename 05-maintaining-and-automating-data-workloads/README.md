# Section 5: Maintaining and Automating Data Workloads

**Exam Weight: ~15%**

Covers operational excellence for data platforms: automating recurring jobs, managing
compute capacity, and ensuring observability for monitoring and troubleshooting.

## Subsections

| Subsection | Topic | File |
|------------|-------|------|
| 5.2 | [Designing automation and repeatability](./5.2-designing-automation-and-repeatability.md) | Scheduling and orchestrating repeatable jobs |
| 5.3 | [Organizing workloads based on business requirements](./5.3-organizing-workloads.md) | Capacity management, BigQuery Editions and reservations |

> **Note:** Subsection 5.1 is not included in the renewal exam guide.

## Key Themes

- **Repeatability**: Idempotent, scheduled jobs that produce consistent results
- **Capacity planning**: BigQuery Editions (on-demand vs. reservations) and slot management
- **Observability**: Monitoring pipeline health, query performance, and data freshness
- **Troubleshooting**: Using logs, metrics, and admin tools to diagnose failures
